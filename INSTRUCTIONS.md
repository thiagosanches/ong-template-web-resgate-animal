# INSTRUCTIONS

You'll develop a static web application, that will be used for a ONG (NGO) for dogs/cats/animals rescue. The person that will maintain the application will populate the data as in json format. There should be a data folder where it will have all the data that the site will present, under data folder you can add: animals and products, the products will hold some data related to their shopping, which the client will click on the items and it will redirect to Whatsapp (there is no need to integrat any paymay gateway).

The project should have data folder, scripts folder (to hold js files) and a styles folder for CSS, since the person will change the style later, for styling let's put a modernized design that people are using out there.

The website will be hosted under github web pages, so it should be a static website, no backend is needed. The data will be in json format, and the client will update the data as needed pushing the changes to the repository.
